# 概要
現在、J1リーグに対応しており、各試合の選手や監督、審判
それぞれを評価やコメントが行えます。

また、他の人の評価も閲覧することができ、お気に入りのユーザーを見つけることができます。

他の人の評価を見ることにより、共感や新しい視点が増え、よりサッカー観戦が楽しくなっていくことを目的としています。
